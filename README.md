# Gerenciador de Objetos em JavaScript

## Descrição
Este projeto é um **gerenciador simples de objetos** feito em JavaScript com Node.js.  
Ele permite criar, armazenar e manipular diferentes tipos de objetos: **pessoas, carros, livros e alunos**.  
Além disso, inclui uma **calculadora básica** e funções para calcular a **média das notas de alunos**.  

O objetivo é praticar conceitos fundamentais de JavaScript, como:
- Objetos e propriedades
- Arrays de objetos
- Funções que manipulam objetos
- Objetos aninhados
- Métodos dentro de objetos
- Iteração sobre objetos

---

## Funcionalidades
- Adicionar pessoas com nome, idade, profissão e hobby  
- Adicionar carros com marca, modelo, ano, potência e tipo  
- Adicionar livros com título, autor e ano  
- Adicionar alunos com notas em português, matemática e história  
- Calcular a média das notas de um aluno  
- Utilizar uma calculadora com métodos de soma e subtração  

---

## Tecnologias
- JavaScript (ES6)
- Node.js (sem dependências externas)
